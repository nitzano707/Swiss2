# אתר הטיול — שוויץ ואיטליה 2026

אתר סטטי, Mobile-First, עובד אופליין (PWA). כל התוכן נמצא בקובץ `data/trip.json` — אפשר לערוך שעות, כתובות וחניות שם בלי לגעת בעיצוב.

## מה חדש
- **תמונות ותיאורים** לאתרים המרכזיים במסלול (בעמוד היום המתאים) — כל התמונות מ-Wikimedia Commons עם רישיון פתוח (CC/Public Domain) וקרדיט מקושר, כדי לא לפגוע בזכויות יוצרים.
- **מזג אוויר חי** (`weather.html`) — תחזית אמיתית בזמן אמת + 10 ימים קדימה לכל אחד מ-4 אזורי הלינה, דרך Open-Meteo (API חינמי, ללא מפתח). נשמר במכשיר אחרי כל טעינה מוצלחת, כך שגם בלי אינטרנט מוצגת התחזית האחרונה שנטענה.
- **מחשבון המרת מטבע** (`currency.html`) — CHF/EUR ל-₪, עם שערים חיים מ-Frankfurter API (מבוסס ECB), ניתנים גם לעריכה ידנית ונשמרים במכשיר.
- **הגדלת/הקטנת טקסט** — שני כפתורים (א-/א+) בראש כל עמוד, לנוחות קריאה בטלפון. הבחירה נשמרת במכשיר.

## איך מעלים ל-GitHub Pages

1. יוצרים repo חדש ב-GitHub (אפשר Private).
2. מעלים את כל תוכן התיקייה הזו (כולל `.nojekyll` אם רוצים למנוע עיבוד Jekyll — לא חובה כאן).
3. **Settings → Pages → Source**: לבחור את הענף (`main`) ותיקיית השורש (`/`).
4. אחרי דקה־שתיים האתר יהיה זמין בכתובת `https://USERNAME.github.io/REPO-NAME/`.
5. בטלפון: לפתוח את הכתובת, ואז "הוסף למסך הבית" (iOS: כפתור השיתוף → הוספה למסך הבית; אנדרואיד: תפריט → התקן אפליקציה). זה מה שהופך אותו ל-PWA שעובד גם בלי אינטרנט.

חשוב לפתוח את האתר **פעם אחת עם אינטרנט** לפני הטיול — כדי שה-Service Worker ישמור את כל הדפים והנתונים במכשיר.

## מה נשמר רק במכשיר (לא עולה ל-GitHub)

- קודי כניסה למלונות (דף מלונות — שדה "קוד כניסה / הערה פרטית")
- סימוני "הושלם" לימים ולתחנות
- הערות אישיות בכל יום
- בחירת תוכנית A/B/C ביום הגמיש (29.7)
- מצב תצוגה בהיר/כהה
- גודל הטקסט שנבחר

כל אלה נשמרים ב-localStorage של הדפדפן — פרטיים לחלוטין, לא נשלחים לשום מקום.

## עדכון תוכן

לערוך את `data/trip.json` (שעות, כתובות, מלונות, חניות, מידע). לאחר שינוי, **חשוב להעלות גרסה חדשה ל-Service Worker** — לפתוח את `service-worker.js` ולהעלות את `CACHE_VERSION` (למשל מ-`"v1"` ל-`"v2"`), אחרת מכשירים שכבר שמרו את האתר אופליין ימשיכו להציג את הגרסה הישנה.

## מה נשאר להשלים לפני הטיול (ר' גם דף "מידע" באתר)

- תוכנית סופית ומדויקת לכל יום, ושעות מדויקות לכל עצירה
- כתובות Waze מאומתות סופית
- חניוני גיבוי נוספים
- תוכניות גשם סופיות לכל הימים
- שעות סופיות לרכבלים, שיט ואתרים
- תשובת מלון ציריך לגבי צ'ק־אין מוקדם
- סטטוס הווינייטה השווייצרית
- טרמינל היציאה המדויק של טיסת החזור
- מחירים ושעות עדכניות לחניונים ואטרקציות
- קישורים סופיים לכל הזמנה
